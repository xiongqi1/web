README.TXT for the Mbedthis HTTP Server.
--------------------------------------------------------------------------------

The Appweb modules directory contains the extension modules for the Appweb
server. These modules may be builtin or may be compiled as loadable modules
depending on the settings in config.make.

See the FILES.TXT for a description of the files and directories.
