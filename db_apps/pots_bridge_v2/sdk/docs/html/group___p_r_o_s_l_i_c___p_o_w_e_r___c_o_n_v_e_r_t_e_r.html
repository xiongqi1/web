int ProSLIC_PowerDownConverter ( proslicChanType_ptr  hProslic)

Safely power down dcdc converter after ensuring linefeed is in the open state. Test power down by setting error flag if detected voltage does no fall below 5v.

Parameters

Return values
int- error from errorCodeType RC_NONE indicates no error.

int ProSLIC_SetDCDCInversionFlag ( proslicChanType_ptr  hProslic,
uInt8  flag 
)

Set channel flag to indicate that the polarity of the DC-DC converter drive signal is to be inverted from what is indicated in the General Parameters. This inversion is relative to the polarity established by the General Parameters, not an absolute polarity.

Parameters
[in]hProslic- which channel to change state.
[in]flag- inversion flag (1 - inverted, 0 - no inversion)
Return values
int- error from errorCodeType RC_NONE indicates no error.
