int ProSLIC_dbgSetRinging ( proslicChanType *  pProslic,
ProSLIC_dbgRingCfg *  ringCfg,
int  preset 
)

Provision function for setting up Ring type, frequency, amplitude and dc offset and to store them in the given preset.

Note
: This function calculates the values, it does not set them.
: This code assumes ProSLIC_RingSetup() was called earlier with the highest ringer requirements (possibly during initialization). After that, this function can be used for meeting provisioning requirements.
Parameters
[in]pProslic- which channel to modify
[in]ringCfg- how to configure the channel's ringer
[in]preset- where to store the new configuration.

int ProSLIC_RingStart ( proslicChanType_ptr  hProslic)

Starts the ringer as confgured by ProSLIC_RingSetup . If the active and inactive timers are enabled, the ProSLIC will automatically turn on/off the ringer as programmed, else one will need to manually implement the ring cadence. In either case, one will need to call ProSLIC_RingStop to stop the ringer or one may call ProSLIC_SetLinefeedStatus to change the state directly.

Parameters
[in]hProslic- which channel to enable ringing.
Return values
int- error from errorCodeType RC_NONE indicates no error.
Note
This function can be compiled out by defining DISABLE_RING_SETUP
Deprecated:
  • Will become a macro in a future release and later removed. Customers should just call ProSLIC_SetLinefeedStatus w/ LF_RINGING.
See also
ProSLIC_dbgSetRinging ProSLIC_RingStop ProSLIC_RingSetup
