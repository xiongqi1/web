int ProSLIC_testLoadTest ( SiVoiceChanType_ptr  pProslic,
BOOLEAN  is_testStart,
uint32_t  timeDelayMsec 
)

Provide a simulated offhook using the ProSLIC test load. You would call this function twice: once to enable the simulated test load (is_testStart = TRUE) and check if the upper layer correctly detected the hook state change and possibly pass some audio. After you have completed this test/check, you would call the same function again to restore the system to normal state (is_testStart = FALSE).

Note
This test can only be executed one at a time. Running this on multiple channels that have different DC Feed and digital control settings will cause problems. If the all the channels have the same settings, this restriction can be loosened.
Parameters
[in]pProslic- channel data structure
[in]is_testStart- is the the start or end of test.
[in]timeDelayMsec- how may mSec to delay prior to returning from the function.
Return values
int- error from errorCodeType
RC_NONEindicates no error.
RC_UNSUPPORTED_FEATUREindicates feature not supported on this device
