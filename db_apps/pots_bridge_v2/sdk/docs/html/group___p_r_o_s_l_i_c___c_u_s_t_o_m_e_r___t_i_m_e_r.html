Detailed Description

This section has documentation related to timers: delays and timestamps. The void * is a customer supplied timer data structure that the user specifies in SiVoice_setControlInterfaceTimerObj. These functions need to be associated with the functions documented in SiVoice Timer functions

Note
For the majority of implementations, the only function that is required is the delay function.

typedef int(* system_timeElapsed_fptr) (void *hTimer, void *startTime, int *timeInMs)

System time elapsed function pointer.

This function combined with system_getTime_fptr function allow one to determine the time elapsed between the two event times.

The example pseudo code below shows how one may see this function used in conjunction with the function pointed by system_getTime_fptr to do some processing while waiting for a time to expire.

1 my_timer_obj time_now;
2 
3 getTime(my_prosclic_obj->timer_obj_ptr, (void *)(&time_now));
4 
5  Some events occur here...
6 
7 do
8 {
9  time_elapsed(my_proslic_obj->timer_obj_ptr,(void *)(&time_now), &elapsed_time_mSec);
10 
11  Do something here...
12 
13  }while(elapsed_time_mSec < Desired delay);
14 
15 Change state...
Parameters
[in]*hTimer- the system timer object
[in]*startTime- the time object returned by system_getTime_fptr - the "start time" of the event.
[out]*timeInMs- the time in mSec between the "start time" and the current time.
Return values
int- error from errorCodeType RC_NONE indicates no error.

system_getTime_fptr
