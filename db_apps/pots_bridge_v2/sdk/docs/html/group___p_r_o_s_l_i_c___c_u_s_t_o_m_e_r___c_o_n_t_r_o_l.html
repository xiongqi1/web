Detailed Description

This group of function pointer prototypes need to be implemented by the customer in order for the ProSLIC API to function correctly. These functions need to be associated with the API by the functions documented in SiVoice SPI/GCI access routines

typedef ramData(* ctrl_ReadRAM_fptr) (void *hCtrl, uInt8 channel, uInt16 ramAddr)

RAM read function pointer.

Parameters
[in]*hCtrl- which interface to communicate through (this is a customer supplied structure)
[in]channel- ProSLIC channel to read from (this is the value passed is the same as found in SiVoice_SWInitChan)
[in]ramAddr- Address of the RAM location to read from
Return values
ramData- the value read. If an error occurs, this value is undefined.
