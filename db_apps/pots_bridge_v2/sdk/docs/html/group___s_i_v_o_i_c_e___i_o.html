Detailed Description

This group of functions are used to associate the transport mechanism (SPI, GCI) functions with the API. The actual functions being references are normally implemented by the customer for their particular OS and platform.

int SiVoice_setControlInterfaceCtrlObj ( SiVoiceControlInterfaceType *  pCtrlIntf,
void *  hCtrl 
)

Associate a interface object with a user supplied data structure. This structure is passed to all the I/O routines that the ProSLIC API calls.

Parameters
[in,out]pCtrlIntf- which interface to associate the user supplied structure with.
[in]*hCtrl- the user supplied structure that is passed to the IO functions.
Return values
int- error from errorCodeType RC_NONE indicates no error.
