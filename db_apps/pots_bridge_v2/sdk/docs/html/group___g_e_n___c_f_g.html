#define PROSLIC_BCAST_RESET   0x8000

Implement soft reset. For dual channel devices, you can specify a hard reset (which impacts both channels) or if the channel is actually the second channel. The default is to do a soft reset on the 1st channel.

Parameters
[in]hProslic- which channel to reset
[in]resetOptions- PROSLIC_HARD_RESET (dual channel devices ONLY) | PROSLIC_SOFT_RESET_SECOND_CHAN (dual channel devices) | PROSLIC_BCAST_RESET | PROSLIC_SOFT_RESET
Return values
int- error from errorCodeType RC_NONE indicates no error.
Note
Will abort reset if in free run mode.
Only perform a soft reset on 1 channel at a time.
