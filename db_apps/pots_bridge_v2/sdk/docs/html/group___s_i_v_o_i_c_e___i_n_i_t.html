int SiVoice_Reset ( SiVoiceChanType_ptr  pChan)

This function calls the user supplied reset function to put and take out the channel from reset. This is typically done during initialization and may be assumed to be a "global" reset - that is 1 reset per daisychain vs. 1 per device.

Note
This function can take more than 500 mSec to complete.
Parameters
[in]pChan- which channel to reset, if a "global reset", then any channel on the daisy chain is sufficient.
Return values
int- error from errorCodeType RC_NONE indicates no error.

This function initializes the various channel structure elements. This function does not access the chipset directly, so SPI/GCI does not need to be up during this function call.

Parameters
[in,out]hProslic- which channel to initialize.
[in]channel- Which channel index is this. For example, for a 4 channel system, this would typically range from 0 to 3.
[in]chipType- chipset family type for example SI3217X_TYPE or SI3217X_TYPE
[in]pDeviceObj- Device structure pointer associated with this channel
[in]pCtrlIntf- Control interface associated with this channel
Return values
int- error from errorCodeType RC_NONE indicates no error.
