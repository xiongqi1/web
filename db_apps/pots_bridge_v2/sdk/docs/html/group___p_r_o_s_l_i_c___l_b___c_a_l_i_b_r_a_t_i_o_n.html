int ProSLIC_LBCal ( proslicChanType_ptr *  pProslic,
int  size 
)

Run canned longitudinal balance calibration.

Parameters
[in]pProslic- start channel to start from for calibration.
[in]size- number of channels to calibrate
Return values
int- error from errorCodeType RC_NONE indicates no error.
Note
Use the ProSLIC_Init function to perform all calibrations except Longitudinal Balance (LB). Use the ProSLIC_LBCal function to perform LB calibration. LB Calibration typically needs to execute LB calibration once per system setup without a load connected. The LB calibration results can be stored in non-volatile storage (flash, eeprom) and retrieved during your normal ProSLIC initialization.
