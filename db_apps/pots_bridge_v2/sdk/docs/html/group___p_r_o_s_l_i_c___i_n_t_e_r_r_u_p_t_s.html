int ProSLIC_GetInterrupts ( proslicChanType_ptr  hProslic,
proslicIntType *  pIntData 
)

This function reads the interrupts from the ProSLIC and places them into a proslicIntType structure which contains up to MAX_PROSLIC_IRQS interrupts.

Parameters
[in]hProslic- which channel to enable the interrupts.
[in,out]pIntData- this structure contains the interrupts pending and the number of interrupts.
Return values
int- number of interrupts pending or RC_CHANNEL_TYPE_ERR if wrong chip type or 0 for no interrupts.
