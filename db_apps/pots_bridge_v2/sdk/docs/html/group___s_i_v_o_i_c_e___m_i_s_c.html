int SiVoice_IdentifyChannels ( SiVoiceChanType_ptr *  pProslic,
int  size,
int *  slicCount,
int *  daaCount 
)

This function probes an array of channel pointers to determine the number of ProSLIC's,if not NULL and DAA's, if not NULL. Should only be called after a reset.

Parameters
[in,out]pProslic- an array of channel pointers to iterate through. ChannelType is modified.
[in]size- number of channels to pobe.
[out]slicCount- if not NULL, the number of SLIC's detected.
[out]daaCount- if not NULL, the number of DAA's detected.
Return values
RC_NONEif successful.
