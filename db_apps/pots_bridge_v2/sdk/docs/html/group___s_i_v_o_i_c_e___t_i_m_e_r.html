Detailed Description

This group of functions associates the customer supplied timer routines with the ProSLIC API. Please note, for most applications, only the delay routine is required. The other routines are needed for the pulse digit API.

int SiVoice_setControlInterfaceGetTime ( SiVoiceControlInterfaceType *  pCtrlIntf,
system_getTime_fptr  getTime_fptr 
)

Description: Associate a time get function with a given control interface. The time get function returns a value in a form of a void pointer that is suitable to be used with the function specified in SiVoice_setControlInterfaceTimeElapsed . This is typically used as a timestamp of when an event started. The resolution needs to be in terms of mSec.

Parameters
[in]pCtrlIntf- which control interface to associate the function with.
[in]getTime_fptr- the pointer to the get time function.
Return values
int- error from errorCodeType RC_NONE indicates no error.
